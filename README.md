# To Loan or Not To Loan 借貸信用評估

## 簡介

1. 目的：金融機構對於缺乏傳統信用紀錄、財務資料等數據的新客戶，如應屆畢業生、新創者等人，難以套用內部評估系統而無法進行有效率放貸決策。若能透過客戶提供的背景資訊衡量客戶的信用，便能使金融機構能快速評估並拓展低放貸風險的新客戶。
2. 使用資料集 : Home Credit Default Risk | Kaggle
